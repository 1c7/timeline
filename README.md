## 时间轴小助手
说明：这是一个做到一半放弃掉的 Electron.js 项目。是个字幕的时间轴编辑。
更具体的说明可以看博客: https://1c7.me/timeline-tools/   

废弃原因：视频做不到 frame by frame 以及毫秒级控制。是 Web Standard 还不够造成的。
